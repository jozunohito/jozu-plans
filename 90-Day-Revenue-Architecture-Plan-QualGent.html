The Plan

90 Days to a Working Sales Engine

JEFF
ARAGON
Phase 1: Days 1-30
Foundation - Carry a Bag, Build the Blueprint
✓ Close deals personally. Start by carrying a bag. Run discovery, demos, and negotiations on live enterprise opportunities to learn the product, the buyer, and the sales cycle from the inside. No one builds a system for a sale they have not personally closed.
✓ ICP segmentation. Define the two primary segments: enterprise (Fortune 500 with deployed voice agents hitting quality walls) and growth-stage (Series A-C companies shipping AI agents fast with no verification layer). Map the buyer personas: Director of Engineering, VP QA, CTO.
✓ Messaging architecture. Build the core message connecting verification to the economic consequence of shipping unverified AI agents: 40% conversation failure rates, customer churn, engineering rework. Every outbound email, every demo opener, every proposal ties back to this.
✓ CRM and pipeline architecture. Set up clean pipeline stages, dashboards, and reporting. Define exit criteria for every stage. Build the data hygiene process that makes forecasting possible.
Phase 2: Days 31-60
Execution - Build the Outbound Engine
✓ Outbound system launch. Architect and launch sequenced outbound campaigns targeting engineering and QA leaders at companies deploying voice and chat AI. Test messaging, iterate on reply rates, and build a repeatable pipeline generation system from first principles.
✓ Coordinate with GTM Engineers. Work with the existing GTM automation team to integrate outbound with automated systems. Ensure the human-led and machine-led motions reinforce each other, not compete.
✓ Enterprise deal execution. Run the full sales process on active enterprise opportunities: discovery, technical demo, ROI case, negotiation, close. Build the repeatable closing motion as deals progress.
✓ Expand existing accounts. Map expansion opportunities within current Fortune 500 customers. Build the account expansion playbook: new business units, additional use cases, increased volume.
Phase 3: Days 61-90
Scale - From Founder-Led to Self-Sustaining
✓ Pipeline generation system running. Outbound producing predictable monthly meetings across enterprise and growth segments. Hit rates improving through constant testing and iteration. The engine runs without founder involvement.
✓ Multiple deals closed. New enterprise and mid-market contracts signed. Revenue trajectory validates the GTM model and provides data for the next phase of scaling.
✓ Canonical GTM playbook written. The complete, documented playbook: ICP definitions, outbound sequences, discovery framework, demo script, objection handling, competitive positioning, closing motion. Every future hire onboards from this document.
✓ Hiring plan defined. Based on 90 days of real data, define the sales team structure: who to hire next (SDR, AE, or both), what profile works, what ramp looks like. Begin recruiting.
