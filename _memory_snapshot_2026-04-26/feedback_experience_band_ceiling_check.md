## The rule

**Before recommending Jeff apply to ANY founding-tier or sub-CRO role, scan the JD for the explicit experience band.** Two distinct seat types both go by "Founding GTM / Founding Head of Sales":

**Type A — Senior-veteran founding seat:** "10+ years," "12+ years," "15+ years," "experienced enterprise sales operator," "proven CRO," "scaled prior sales orgs." Jeff is the target.

**Type B — Mid-career founding seat:** "2-4 years," "3-5 years," "5-7 years," "early-career closer," "first sales hire we want to grow with." Jeff is **structurally over-altitude.** Auto-reject likely on years-of-experience filter regardless of resume variant.

## Why both seat types exist

Some founder-CEOs hire a senior commercial veteran for the founding GTM seat (Alinia, Lovable, Vindexa, Versori archetype). Other founder-CEOs hire a hungry mid-career closer who'll grow with the company and be replaced by a senior CRO at Series C (StackOne archetype). The JD experience band is the signal that distinguishes them.

## The pre-flight check

Before drafting any cover letter or recommending an application:

1. Read the "What you've done" / "Requirements" / "What you'll bring" section of the JD.
2. Find the years-of-experience floor stated.
3. **If floor is single-digit (2 / 3 / 4 / 5 / 6 / 7 years):** Jeff is OVER-altitude. Skip the application OR pivot to one of:
   - Warm-intro through investor / founder network (bypass the years-of-experience ATS filter)
   - Cold-DM founder with senior-advisor framing (not candidate-in-the-band framing)
   - Skip entirely — this is structurally not Jeff's seat
4. **If floor is double-digit (10+ years, 12+ years, etc.):** Jeff is in the target band. Proceed with application.
5. **If floor is unspecified:** Default to applying via senior-veteran framing, but flag the ambiguity and recommend warm-intro path in parallel.

## Sources of failure this prevents

- StackOne 2026-04-24 — JD said "2-4+ years," Jeff applied as 25-year veteran, auto-rejected within ~24h. Different failure mode than wrong-variant or vertical-mismatch.
- Future YC W26 / S25 / S24 founding seats where the founder wants a hungry first-AE-grown-into-leader, not a senior veteran.
- Any "founding GTM" role where the JD is implicitly written for a 5-7 year operator.

## How agents implement this rule

When an agent (K1-K14, jozu-founding-tier-global-scan, etc.) surfaces a role for Jeff:

1. Read the JD's experience band requirement.
2. If single-digit years → tag the role as "OVER-ALTITUDE FILTER RISK" and recommend warm-intro / cold-DM / skip rather than direct application.
3. If double-digit years → tag as "IN BAND" and recommend direct application.
4. If unspecified → tag as "AMBIGUOUS — verify before draft."
