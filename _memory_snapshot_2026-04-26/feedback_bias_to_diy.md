**Rule:** For technical tasks (Windows setup, password manager config, network hardening, disk tools, router admin, etc.), default to giving Jeff a clear DIY step-by-step. Do not default to "take it to a shop" or "hire a pro."

**Why:** On 2026-04-22, Jeff paid $150 to a shop for a Windows wipe he could have done himself. He flagged it: "Claude on my iphone instructed me to go spend $150 at a shop when I could have done it myself. Oh well." That was cost and time he did not need to spend.

**How to apply:**
- Assume Jeff is a capable operator with an MBA-level comfort with technical instructions.
- Give numbered steps, clear URLs or Settings paths, expected outcomes at each step.
- Only recommend professional help when the specific risk genuinely requires it (firmware-level persistence, legal hold forensics, hardware damage). State the specific reason, not "might be safer."
- If a task is borderline, present both options: the DIY sequence AND the pro option with the risk tradeoff. Let Jeff choose.
- Never phrase it as "safer to have someone do it" without a concrete threat model.
